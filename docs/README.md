# PayCore Docs

Dokumentasi PayCore dipisahkan berdasarkan pembaca.

## Untuk agen di luar PayCore

Baca `docs/external/` ketika Anda sedang bekerja dari repo aplikasi lain
seperti Narraza, Siklusio, TEKAD, atau produk AppVibe lain yang ingin
integrasi dengan PayCore.

Mulai dari:

- `docs/external/integration-guide.md`
- `docs/external/openapi.yaml`
- `docs/external/app-authentication.md`
- `docs/external/payment-events.md`
- `docs/external/examples/generic-app-integration.md`
- `docs/external/troubleshooting.md`

## Untuk agen internal PayCore

Baca `docs/internal/` ketika Anda sedang mengubah, deploy, mengoperasikan,
atau menambah konfigurasi di repo PayCore.

Mulai dari:

- `docs/internal/architecture.md`
- `docs/internal/deployment.md`
- `docs/internal/runbook.md`
- `docs/internal/integrating-new-app.md`
- `docs/internal/duitku-integration.md`
- `docs/internal/duitku-sandbox-setup.md`
- `docs/internal/staging-e2e-checklist.md`

## Catatan

Dokumen eksternal adalah kontrak integrasi yang aman dibaca agen repo lain.
Dokumen internal dapat berisi detail operasi, deployment, D1, provider, dan
runbook yang hanya relevan untuk maintainer PayCore.
