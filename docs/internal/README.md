# Internal PayCore Docs

Folder ini untuk agen dan maintainer yang bekerja langsung di repo PayCore.

Gunakan dokumen berikut sesuai tugas:

- `architecture.md` - gambaran status, persistence, dan boundary sistem.
- `deployment.md` - deploy, binding, secret, domain, dan readiness production.
- `runbook.md` - retry fulfillment, manual review, dan operasi harian.
- `integrating-new-app.md` - checklist internal untuk mendaftarkan app baru.
- `duitku-integration.md` - detail provider Duitku di sisi PayCore.
- `duitku-sandbox-setup.md` - setup sandbox/operator staging.
- `staging-e2e-checklist.md` - checklist E2E staging.
- `test-plan.md` - quality gate dan skenario test.
- `examples/` - contoh flow internal/operator.
- `superpowers/` - spesifikasi dan rencana kerja internal.

Untuk kontrak yang harus dibaca agen repo lain, gunakan `docs/external/`.
